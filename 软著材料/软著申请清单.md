## 二、软件功能简介（200–500 字版本）

**嗓音声学品质多维分析图谱（VoiceMap）** 是一款面向嗓音临床筛查、
歌手训练和声乐教学的桌面应用软件。给定一段立体声 WAV 录音
（通道 1 为麦克风信号，通道 2 为电声门图 EGG 信号），软件对录音
做逐周期切分，在「MIDI 音高 × 声压级（SPL）」二维网格上聚合输出
40 余项嗓音质量指标，并实时渲染 Voice Range Profile (VRP) 热图。

**核心功能**

1. **声学指标计算**：CPP、CPPS、HNR、NHR、Jitter（local/RAP/PPQ5）、
   Shimmer（local/dB/APQ3/APQ5/APQ11）、PPE、ZCR、谱质心、谱带宽、
   谱滚降、谱平坦度、Alpha Ratio、Hammarberg Index 等共 25+ 项。
2. **电声门图（EGG）指标**：Qcontact、Icontact、dEGGmax、HRFegg、
   OQ、SPQ、CIQ 等共 7 项；基于 KTH FonaDyn 项目算法。
3. **唱歌专属指标**：Vibrato 速率 / 幅度、共振峰 F1/F2/F3、共振峰
   带宽 B1/B2/B3、Singer's Formant、H1H2、H1H3 等 11 项。
4. **聚类分析**：EGG 周期 K-means 聚类（k=5 默认），输出 maxCluster
   主导簇 + Cluster 1~5 占比；嗓音质量 cPhon 聚类。
5. **VRP 热图渲染**：每项指标一张 64 × 81 网格的色彩图，色彩学院
   采用感知均匀（perceptually uniform）+ 色盲安全调色板。
6. **临床范围参考**：依据 MDVP、KayPENTAX、Praat 等公开文献阈值，
   对每个指标自动分级（健康 / 中等 / 注意 / 异常）。
7. **多格式导出**：CSV、Excel 多 sheet、Markdown 临床叙述报告、
   PNG / PDF / SVG / EMF 图片。
8. **跨录音对比**：同时加载两段录音，输出 A | B | A−B 三联图。
9. **中英文双语界面**：动态切换，无需重启。
10. **PyInstaller 单文件分发**：终端用户无需安装 Python。

## 三、软件技术特点（200–500 字版本）

1. **基于科学文献的算法实现**：所有声学公式直接复现 MDVP、Praat、
   Hillenbrand、McLeod-Wyvill NSDF、KTH FonaDyn 等公开发表的标准
   方法，保证数据可与同领域研究横向对比。
2. **统一的 Metric Registry 架构**：50+ 项指标元数据（单位、阈值、
   色板、参考范围）集中维护在 `metrics_registry.py`，下游所有消费者
   （GUI 下拉、绘图、Excel、临床报告）都从 registry 读取，新增指标
   无需改多个文件。
3. **可访问性设计**：界面通过 WCAG 2.1 AA 文字对比度审计；所有
   交互元素 Tab 键可达且有 2 px 焦点环；颜色编码均通过红绿色盲
   仿真验证。
4. **本地化双语**：基于 dict-based STRINGS 表的 i18n 体系，所有
   UI 字符串、tooltip、错误提示同时维护中文与英文版本。
5. **响应式 GUI**：分析在后台线程执行，主线程负责 UI 渲染；
   matplotlib motion 事件支持 60~120 Hz 鼠标 hover 实时读取。
6. **跨录音聚类一致性**：支持保存 / 加载 K-means 中心点 CSV，
   保证不同录音的 cluster 1~5 标签语义一致。
7. **批处理**：CLI 模式支持整目录批处理，可指定参数模板。
8. **打包独立分发**：PyInstaller one-folder 模式打出 ~17 MB 主程序
   + 依赖，提供 Inno Setup 安装包 `VoiceMap_v1.0.0_setup.exe`。
