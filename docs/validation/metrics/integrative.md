# MPT / Voicing Ratio / DUV — integrative metrics

> Three whole-recording summaries, each computed once and broadcast to every
> (MIDI, dB) cell so they still appear as a (uniform) layer in the VRP grid:
> - **MPT** — Maximum Phonation Time proxy: longest contiguous voiced run (s);
> - **VoicingRatio** — voiced cycles / total analysed cycles (0–1);
> - **DUV** — Degree of Unvoiced = 100·(1 − VoicingRatio) (%).

## 1. Implementation
- File: `voicemap/metrics.py` → `IntegrativeMetricsCalculator` (lines
  1568–1610).
- A cycle is voiced iff `midi_per_cycle > 0`. Cycle durations
  `T[i] = diff(cycle_triggers)/fs`. MPT walks the voiced flags accumulating
  `T` over consecutive voiced cycles and keeps the running maximum;
  VoicingRatio = `voiced.mean()`; DUV = `100·(1 − VoicingRatio)`.
- Registry keys `MPT`, `VoicingRatio`, `DUV` (Density, `待验证=True`).

## 2. Reference Standard
- These are simple, self-defined bookkeeping quantities (no external tool):
  - longest-run length, arithmetic mean of a boolean, and its complement.
- The reference is therefore the **closed-form value** for a designed input,
  recovered to machine precision.
- Clinical context: "Maximum Phonation Time" (Baken & Orlikoff 2000) is a
  max-effort sustained-vowel task; ours is a software proxy (longest voiced run
  in *this* recording) — see §7.

## 7. Known Limitations
- **Not the clinical MPT.** Our MPT is the longest contiguous *voiced run in
  the analysed recording*, so it is **bounded by the recording length** (67 s
  on the 70 s fixture). The clinical Maximum Phonation Time requires a
  max-effort sustained-/a/ protocol and a recording at least as long as the
  subject can phonate — it is a *recording-protocol* measure, not something the
  software can synthesise from an arbitrary clip. Use it as a relative
  "longest steady stretch", not an absolute clinical MPT.
- **VoicingRatio is relative to *analysed* cycles.** The cc / phase-portrait
  cycle-detection front-end already places triggers only in voiced regions, so
  on clean sustained voice VoicingRatio ≈ 1 by construction. It therefore
  mainly flags per-cycle pitch-assignment failures (octave slips, low-clarity
  cycles set to MIDI 0), **not** silence or unvoiced consonants between
  syllables.
- **Broadcast layer.** All three are single scalars painted uniformly across
  the VRP — they carry no per-cell structure; they are recording-level
  summaries surfaced through the grid for convenience.
- **(C) corpus distribution** (healthy vs disordered MPT/voicing ranges)
  deferred — needs a connected-speech / max-effort corpus.
