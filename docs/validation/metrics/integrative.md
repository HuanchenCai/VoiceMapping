# Voicing Ratio / DUV — integrative metrics

> Two whole-recording summaries, each computed once and broadcast to every
> (MIDI, dB) cell so they still appear as a (uniform) layer in the VRP grid:
> - **VoicingRatio** — voiced cycles / total analysed cycles (0–1);
> - **DUV** — Degree of Unvoiced = 100·(1 − VoicingRatio) (%).
>
> (**MPT was removed (2026-06-01)** — a whole-recording scalar with no per-cell
> meaning, and not a true clinical Maximum Phonation Time; see §7.)

## 1. Implementation
- File: `voicemap/metrics.py` → `IntegrativeMetricsCalculator`.
- A cycle is voiced iff `midi_per_cycle > 0`. `VoicingRatio = voiced.mean()`;
  `DUV = 100·(1 − VoicingRatio)`. Both are broadcast to every cycle.
- Registry keys `VoicingRatio`, `DUV` (Density).

## 2. Reference Standard
- Simple, self-defined bookkeeping quantities (no external tool): the arithmetic
  mean of a boolean and its complement.
- The reference is therefore the **closed-form value** for a designed input,
  recovered to machine precision.

## 4. Validation Method
- **(B) Synthetic ground truth** — on the designed grid:
  - `VoicingRatio == voiced/total` (atol 1e-9);
  - `DUV == 100·(1 − VoicingRatio)` (atol 1e-9);
  - corners: all-voiced → (1, 0); all-unvoiced → (0, 100).
- **(C) Real-audio sanity** — on a sustained voiced recording VoicingRatio
  `≈ 1`, DUV identity holds.

## 7. Known Limitations
- **MPT removed.** The old "MPT" was the longest contiguous voiced run in the
  analysed recording — a single scalar broadcast uniformly across the VRP (no
  per-cell meaning) and **not** a clinical Maximum Phonation Time (which needs a
  max-effort sustained-/a/ protocol and a recording at least as long as the
  subject can phonate). It was dropped from the pipeline, registry, CSV/Excel,
  GUI, report thresholds, and i18n.
- **VoicingRatio is relative to *analysed* cycles.** The cc / phase-portrait
  cycle-detection front-end already places triggers only in voiced regions, so
  on clean sustained voice VoicingRatio ≈ 1 by construction. It therefore mainly
  flags per-cycle pitch-assignment failures (octave slips, low-clarity cycles
  set to MIDI 0), **not** silence or unvoiced consonants between syllables.
- **Broadcast layer.** Both are single scalars painted uniformly across the VRP
  — they carry no per-cell structure; they are recording-level summaries
  surfaced through the grid for convenience.
- **(C) corpus distribution** (healthy vs disordered voicing ranges) deferred —
  needs a connected-speech corpus.
