# SFE / SPR — Singer's-formant band measures

> Energy in the **singer's formant** region (~2.8–3.4 kHz, the "ring" of a
> trained classical voice). **SFE** = singer-band energy relative to total;
> **SPR** = high-band vs low-band energy. Both rise with a strong singer's
> formant.

## 1. Implementation
- File: `voicemap/metrics.py`
  - **SFE** — `FormantCalculator` (lines ~1408–1445): per 25 ms frame
    (pre-emphasis 0.97, Hamming), FFT power; `SFE = 10·log10( Σ power[2.8–3.4
    kHz] / Σ power )`.
  - **SPR** — `FormantExtrasCalculator` (lines ~1504–1522): same framing;
    `SPR = 10·log10( Σ power[2–4 kHz] / Σ power[0–2 kHz] )`.
- Both assign the frame value per cycle.

## 2. Reference Standard
- **SFE**: Sundberg (1974) — the singer's formant. **SPR**: Omori et al.
  (1996) — Singing Power Ratio. The canonical evidence is a *corpus*
  difference (trained classical singers show a strong singer's formant;
  untrained speech does not) — that is the **(C)** test, needing a singer
  corpus.
- For **(B)** we use synthetic tones with controlled band energy: the
  band-energy ratio has a known sign/shape, so SFE must be monotonic in
  singer-band energy and SPR linear in the hi/lo ratio.

## 3. Test Signals
Synthetic (in-harness, F0-trigger grid 150 Hz):
- SFE: a fixed 500 Hz (out-of-band) tone + a 3000 Hz (in-band) tone whose
  amplitude is swept `{0, 0.5, 1, 2}`.
- SPR: 1000 Hz (low band) + 3000 Hz (high band) at ratios `{0.25 … 4}`.

## 4. Validation Method
- **(B) Synthetic ground truth:**
  - SFE rises **monotonically** as singer-band energy increases; `≪ 0 dB`
    with none, `≈ 0 dB` when the band dominates;
  - SPR is **linear** in `20·log10(A_hi/A_lo)` — slope ∈ [0.95, 1.05],
    Pearson r ≥ 0.99.
- **(A)** no reference tool. **(C)** the classical-singer-vs-speech corpus
  test is deferred (needs a singer corpus; the synthetic GT meets the P1 bar).

Reproducible steps:
```bash
python scripts/validate_metric.py singer_formant
```

## 7. Known Limitations
- **Pre-emphasis offset.** Both apply a 0.97 pre-emphasis, which boosts the
  high band, so the *absolute* SFE/SPR carry a fixed positive offset vs an
  un-pre-emphasised band ratio. Only the *trend* (SFE monotonicity, SPR
  slope-1) is intrinsically validated; re-derive absolute cut-offs on
  VoiceMap's own pipeline.
- **Fixed bands.** SFE uses 2.8–3.4 kHz; a singer's formant centred slightly
  outside (voice type, sex) is under-counted. SPR's 2 kHz split is a coarse
  hi/lo divide.
- **Discrimination not yet shown on a corpus.** The headline claim — trained
  singers separate from speech — is a (C) corpus test, deferred until a singer
  corpus is wired (VOICED has no singers; SVD has sustained vowels, not
  classical singing).
- **Frame-based**, assigned per cycle; needs correct voicing/cycle marking.
